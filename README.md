## 📖 About This Repository

This repository is a **complete, topic-wise structured collection** of Data Structures and Algorithms implemented in **pure Java**. Every concept is coded from scratch with a focus on:

- ✅ **Clean, readable code** for learning
- ✅ **Interview-ready implementations** (LeetCode / FAANG style)
- ✅ **Coverage from basics to advanced** (Beginner → Expert)
- ✅ **Perfect for revision** before technical interviews

> 💡 Whether you're just starting out or preparing for top-tier product company interviews, this repo has you covered.

## ▶️ How to Run

### Prerequisites

- Java JDK **8 or above** installed
- Any IDE: **IntelliJ IDEA** (recommended) / Eclipse / VS Code

### Steps

```bash
# 1. Clone the repository
git clone https://github.com/CODEWITH-JAIVY/javaDSA-Master-Codewithjaivy.git

# 2. Navigate into the project
cd javaDSA-Master-Codewithjaivy

# 3. Open in your IDE
# (IntelliJ: File → Open → select the folder)

# 4. Navigate to any topic folder and run the .java file
# Example:
cd ARRAY/src
javac ArrayProblems.java
java ArrayProblems
```

## 🔑 Key Algorithms Implemented

- **Sorting:** Bubble Sort, Selection Sort, Insertion Sort, Merge Sort, Quick Sort
- **Searching:** Linear Search, Binary Search, Search in Rotated Array
- **Tree Traversals:** Inorder, Preorder, Postorder, Level Order (BFS)
- **Graph Algorithms:** BFS, DFS, Adjacency List (Undirected Graph)
- **DP Problems:** Fibonacci, Memoization, Tabulation
- **Linked List:** Reverse, Detect Cycle, Merge, Find Middle
- **String:** Anagram, Substring, Palindrome, String Builder
